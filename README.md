Cellular
=========

Let's build a spreadsheet!

## Tokenizer

Turns `SUM(A1, B3)` into a list of tokens.

## AST

Builds an abstract syntax tree from a list of tokens.

## Interpreter

Executes the AST.

## Evaluation order

Determine the order of cell evaulation.

## REPL

```bash
yarn repl
```

Example:

```
SUM(A1, B3)
```
